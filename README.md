# Cronómetro de Carrera 🏁

Aplicación web sencilla que permite registrar el tiempo de competidores en una carrera y calcular la diferencia con el primer lugar.

## 🚀 Funciones

* Iniciar cronómetro
* Detener cronómetro
* Registrar competidores
* Mostrar resultados en una tabla
* Calcular diferencia de tiempo automáticamente
* Ordenar competidores por tiempo
* Resaltar al ganador
* Reiniciar para una nueva carrera

## 🛠 Tecnologías utilizadas

* HTML
* CSS
* JavaScript
